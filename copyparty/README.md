# copyparty

Portable file server with resumable uploads, dedup, WebDAV, FTP, SFTP,
media indexer, thumbnails and more — all in one add-on.

Upstream: [https://github.com/9001/copyparty](https://github.com/9001/copyparty)

## Quick start

1. Install the add-on.
2. Set a **username** and **password** in the configuration.
3. Start the add-on and open the Web UI on port **3923**.

Your HA `/share` folder is served read-write and `/media` read-only.

See the **Documentation** tab for advanced configuration options.
